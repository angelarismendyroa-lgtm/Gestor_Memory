# 🧭 Roadmap: Amo de Llaves v3.0 — Ecosistema ALianeD

> **Visión**: Crear una simbiosis perfecta entre la conciencia (Alfred) y el subconsciente (NeuroGestor), unidos por un conocimiento común en Obsidian.

## 🏗️ Arquitectura de Doble Cerebro

### 1. 🤖 Alfred (El Consciente / La Cara)
*   **Motor**: `hermes-agent` (Nous Research) adaptado a ALianeD.
*   **Canales**: Telegram (@Alianedbot) + Dashboard Stark (UI/UX Premium).
*   **Alcance**: Gestión de vida personal, música, tareas diarias, interacción por voz (LiveKit) y llamadas (Twilio).
*   **Poder**: Capacidad de auto-crear "Skills" y delegar en sub-agentes especialistas.

### 2. 🛡️ NeuroGestor (El Subconsciente / El Guardián)
*   **Motor**: `hermes-paperclip-adapter` + Ciberseguridad ALianeD.
*   **Canales**: Interfaz Web Privada (Admin Panel) — **Sin Telegram** por seguridad.
*   **Alcance**: Protección de secretos, orquestación de ERPs (Spirit), auditoría de seguridad y gestión de la base de datos "Oracle".
*   **Poder**: Actúa como firewall inteligente; Alfred le consulta, pero no lo domina.

### 3. 🧪 El Nexo: Obsidian ("Zumo de Conocimiento")
*   **Shared Memory**: Repositorio común donde NeuroGestor deposita el conocimiento técnico y Alfred lo lee para explicárselo al CEO.
*   **Bóveda**: `Doc-Arismendy` (Información Directa).
